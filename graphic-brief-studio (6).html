🔑 שלב 0 — הגדרת מפתח OpenAI

1. היכנס ל-platform.openai.com ← API Keys ← Create new secret key
2. הדבק את המפתח בתיבה בסרגל הצדדי ולחץ "שמור מפתח"
3. ודא שיש יתרת אשראי בחשבון (Billing) — בלי אשראי הקריאות ייכשלו.
המפתח נשמר רק בדפדפן שלך (localStorage) ולא נשלח לשום שרת מלבד OpenAI.

✏️ שלב 1 — אפיון הבריף

ככל שתהיה ספציפי יותר — התוצאה טובה יותר: שם מותג + תעשייה נותנים הקשר, הפורמט קובע את יחס התמונה, והסגנון + צבעים מכוונים את האסתטיקה.

⚡ שלב 2 — פרומפט ויצירה

לחיצה על "בנה פרומפט" שולחת את הבריף ל-GPT-4o-mini שמנסח פרומפט מקצועי באנגלית. אפשר לערוך אותו ידנית לפני היצירה, או ללחוץ שוב לגרסה שונה. אחר כך "צור גרפיקה" שולח ל-DALL·E. תוצאה תוך 15–30 שניות.

💡 טיפים להצלחה

  • טקסט בגרפיקה: מודלי תמונה מתקשים בטקסט, במיוחד בעברית — עדיף להוסיף ב-Canva/Figma אחר כך. השאר את התיבה "שלב כותרת" כבויה.
  • לוגו: ציין בהוראות "ללא לוגו" והוסף ידנית.
  • גרסאות: "נסה שוב" 2–3 פעמים ובחר את הטובה.
  • שמירה: הגלריה היא לסשן בלבד — הורד קבצים שאתה רוצה לשמור.

⚠️ שגיאות נפוצות

  • 401 / invalid_api_key: המפתח שגוי או הועתק חלקית.
  • 429 / insufficient_quota: נגמר האשראי בחשבון OpenAI.
  • 403 ב-GPT Image 1: המודל דורש אימות ארגון — עבור ל-DALL·E 3.
  • content_policy_violation: הבריף הכיל תוכן שהמודל מסרב לו — נסח מחדש.
